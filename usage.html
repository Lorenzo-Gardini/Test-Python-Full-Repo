mylibrary.math_func.my_sum(a: float, b: float) → float[source]

Adds two integers and returns the result.

This method takes two integer values, adds them together, and returns the result as an integer.

Parameters:

  • a (float) – The first integer value.

  • b (float) – The second integer value.

Returns:

The sum of a and b.

Return type:

float

Example: >>> my_sum(2, 3) 5
